# Quiz-app
A responsive Quiz App developed using vanilla JavaScript that allows users to answer multiple-choice questions, track scores, and restart the quiz. Designed with a clean and user-friendly interface.

## 📌 Description
This is a simple and interactive Quiz App built using HTML, CSS, and JavaScript. It allows users to answer multiple-choice questions, view their score, and restart the quiz.

---

## 🚀 Features
- 📋 Multiple-choice questions
- 👉 One question displayed at a time
- ✅ Instant answer selection
- 📊 Score tracking
- 🔄 Restart quiz option
- 🎨 Clean and responsive user interface

---

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)
